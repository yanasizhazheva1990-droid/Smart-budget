# Smart Budget
## Бриф проекта

---

## Что это?

Одностраничное веб-приложение для учёта личных финансов и контроля расходов с умной системой лимитов и накоплений. Реализация — один HTML-файл со встроенными стилями и JavaScript, без бэкенда и установки. Готов к размещению на любом статическом хостинге.

---

## Для кого?

Люди 20–35 лет, которые хотят:

- контролировать расходы и не выходить за рамки бюджета
- понимать структуру трат и видеть, где можно экономить
- комфортно копить деньги
- сохранять баланс между экономией и комфортом

### Умный пересчёт лимитов

Если потрачено меньше дневного лимита — остаток перераспределяется на следующие дни. Если больше — лимиты на оставшиеся дни уменьшаются автоматически.

**Пример:** бюджет $900, дневной лимит $30. Потрачено $20 → +$10 перераспределяется. Потрачено $45 → следующие дни получают меньший лимит.

## Страницы приложения

| Страница | Что внутри |
|----------|------------|
| **Обзор (Dashboard)** | Остаток на месяц, потрачено, дневной лимит, пересчитанный лимит, цель накопления, график за 7 дней, категории, последние траты, заметки |
| **Добавить расход** | Сумма, категория, комментарий, дата, кнопка «Сохранить» |
| **Аналитика** | Фильтр по периоду, график по дням, круговой график, список расходов, экспорт в Excel — всё начинает считаться и отображаться сразу после внесения первых трат |
| **Заметки** | Добавить заметку с датой, список всех заметок |
| **Настройки** | Бюджет, цель, валюта, дополнительные бюджеты, категории |

## UX / UI стиль

### Настроение

Минималистичный, спокойный, «дорогой» — clean UI без перегруженности. Ориентир по стилю: Yoch Agency.

### Цвета

| Роль | Код цвета | Описание |
|------|-----------|----------|
| Основной фон | `#A8A882` | Насыщенный оливковый |
| Поверхность | `#F7F5EE` | Тёплый кремовый |
| Акцент | `#E8641A` | Оранжевый |
| Текст основной | `#1A1A14` | |
| Текст вторичный | `#6B6B55` | |
| Успех | `#4A7C59` | Зелёный |
| Ошибка | `#C94040` | Красный |

### Типографика

- **Заголовки:** Cormorant Garamond — тонкий серифный, editorial-стиль
- **Интерфейс:** Jost — геометрический гротеск, чистый

### Что не использовать

- Тёмные фоны
- Glassmorphism и тяжёлые тени
- Сложные анимации и перегруженные экраны
- Агрессивные цвета
- Всплывающие окна

## Адаптивность

- Полная мобильная адаптация (основной сценарий — телефон)
- Учёт safe area iPhone (челка, домашний индикатор)
- Минимальный размер зон касания — 44px
- Инпуты без автозума на iOS (font-size: 16px)
- Стили для экранов ≤360px (iPhone SE, старые Android)

---

## Чего нет в проекте

- Регистрации и авторизации
- Общей базы данных (у каждого пользователя свои данные в его браузере)
- Данные по истечению прошлых месяцев хранятся и не стираются — пока пользователь не удалит их сам
- Бэкенда и сервера
- Платёжных функций
- Push-уведомлений

---

## Готовность

Проект полностью готов к использованию. Файл `finance-tracker.html` открывается в браузере без установки и публикуется на любом статическом хостинге перетаскиванием файла.
